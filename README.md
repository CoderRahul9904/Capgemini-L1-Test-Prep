# Capgemini L1 Test Prep

A comprehensive practice tool designed for Capgemini L1 certification preparation. This interactive quiz application covers the full syllabus including Java 8-21, Spring Boot, JUnit 5, Mockito, and Microservices.

## 🚀 Live Demo
[https://capgemini-l1-test-prep.vercel.app](https://capgemini-l1-test-prep.vercel.app)

## ✨ Features
- **Comprehensive Modules**: Java 8, 9-17, 21, OOPS, Collections, Threads, Exceptions, Spring, Spring Boot, Security, Microservices, JUnit 5, and Mockito.
- **Practice Modes**: 
  - **Module-wise**: Focus on specific topics.
  - **Mixed Quiz**: Random questions from all modules.
  - **Weak Areas**: Dynamically generates a quiz based on topics where your score is below 70%.
- **Persisted Progress**: Automatically saves your scores and streaks to local storage.
- **Detailed Explanations**: Every question includes a detailed explanation and code snippets where applicable.

## 🛠️ Tech Stack
- **Frontend**: Vanilla HTML5, CSS3 (Modern UI), JavaScript (ES6+).
- **Deployment**: Vercel.

## 📂 Project Structure
- `index.html`: The core application containing HTML, CSS, and JS logic.
- `README.md`: Project documentation.
